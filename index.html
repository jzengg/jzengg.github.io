lote :: Github
Simple command-line templating engine in Ruby.

    lote
  • Builds up a stringified proc to be evaluated using 'instance_eval' with JSON data as context
  • Supports nested looping constructs using 'each'
  • Supports flow control keywords like 'if, unless, else, elsif'
  • Tested with RSpec and documented with YARD
