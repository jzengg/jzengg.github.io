Recent Projects

Felamily :: Github
Track pets and adoptions at a shelter

    Felamily
  • Uses Paperclip, AWS S3, and Figaro to store uploaded images in the cloud.
  • Provides visual feedback to users on Ajax requests using Flux architecture
  • Live search only makes 1 query to database and filters the results of that query.
  • Uses custom mixin to autofocus input fields

Snake :: Github
Snake-variant inspired by the original arcade game

    Snake
  • Supports standard play and two-player mode with collision and winner detection, including a special case for head-on collisions.
  • Local highscores are stored using Window.localStorage
  • All game graphics drawn using pure CSS3.

wORM :: Github
An ORM inspired by the functionality of ActiveRecord

    wORM
  • Executes SQL WHERE queries with protection against SQL injection attacks
  • Supports associations between models including belongs_to, has_many, has_many_through and has_one_through using custom SQL queries written in heredocs

About Me

In college, I majored in biology. Along the way, I took classes in many different disciplines, but the ones I liked best were those where I had to solve problems. From biochemistry to abstract algebra, I never got tired of learning about new systems and learning to tackle their challenges. After learning Ruby, I was hooked on programming and the new world of challenges it presented. I've now built games in Javascript, Ruby and one full-stack web app using React.js and Rails. I can't wait to see what lies ahead for me as a web developer.
